```
npm install proyecto-2c-crono
```

## Uso

### Contador
```
const modulo = require('proyecto-2c-crono');

const cont = new modulo.Contador(2);

var d = cont.start().subscribe(
    data =>  {
        console.log(data);
        if (data === 'FINISH') {
            d.unsubscribe();
        }
    }
);
//Espera estos resultados
00:00:00
00:00:01
FINISH

```
